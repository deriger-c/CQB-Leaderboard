# CQB Room Leaderboard

Статический сайт для таблицы лидеров CQB Room с публичным рейтингом, историей последних попыток и встроенной админкой для сотрудников.

## Что входит

- публичный топ по лучшему времени каждого игрока;
- переключатель периода `היום / השבוע / כל הזמנים` (фильтрация на клиенте, без изменений API);
- delta-подсказка под каждым игроком (`מוביל` у лидера, `+0.84s` у остальных) и выделение топ-3 (★ ◆ ●);
- список последних попыток (в рамках выбранного периода);
- подсветка только что добавленного результата и сообщение с занятым местом;
- вход в админку по паролю;
- запоминание доверенного устройства через `localStorage`;
- добавление и удаление последних записей;
- интеграция с Google Sheets через Google Apps Script.

## Как подключить Google Sheets

1. Создайте Google Sheet.
2. Откройте `Extensions -> Apps Script`.
3. Вставьте код из `apps-script/Code.gs`.
4. В `Project Settings -> Script Properties` добавьте:
   - `ADMIN_PASSWORD` — пароль для сотрудников;
   - `SPREADSHEET_ID` — ID вашей Google Sheet;
   - `TRUSTED_DEVICES` — оставьте пустым или не создавайте вручную.
5. Сохраните и задеплойте `Deploy -> New deployment -> Web app`.
6. Для `Who has access` выберите `Anyone`.
7. Скопируйте URL Web App и вставьте его в `config.js` в поле `API_BASE_URL`.

Apps Script сам создаст лист `Attempts` и заголовки:

- `id`
- `player_name`
- `time_ms`
- `time_display`
- `created_at`

## Поведение рейтинга

- каждая попытка сохраняется в истории;
- в лидерборд попадает только лучший результат игрока;
- если времена одинаковые, выше будет более ранняя попытка;
- имя игрока нормализуется через `trim` и схлопывание лишних пробелов;
- период `היום` использует локальную дату пользователя, `השבוע` — последние 7 дней (скользящее окно), `כל הזמנים` — всё; статистика сверху пересчитывается по выбранному периоду.

## Формат времени

Админка принимает:

- `12.45`
- `1:12.45`

На сайте время показывается в едином формате:

- `18.42`
- `1:12.45`
